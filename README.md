Product: Simple C# browser with webview.

Current state: Still in development.

Completely usable. 
UPDATE: before updating to new version, run Update_manager.bat & follow instructions.
INSTALLATION: 
1. DISABLE ANTIVIRUS

2. download **_ONLY_** the release files, containing **_.exe & .pak_**, make sure they are in the same folder & run Horizon.START shortcut. 

3. for straightforward installation download ONLY the .zip file.
executable file location: Horizon_Browser/bin/release
